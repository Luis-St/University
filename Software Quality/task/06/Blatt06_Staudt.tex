\chapter*{Aufgabe 6: Fahrzeugvermietungssystem}

Ein Fahrzeugvermietungssystem prüft, ob ein Kunde berechtigt ist, eine bestimmte Fahrzeugklasse zu mieten.
Es gelten folgende Bedingungen:

\begin{enumerate}
	\item Der Fahrer muss mindestens 21 Jahre alt sein.
	\item Der Fahrer muss mindestens 2 Jahre im Besitz eines Führerscheins sein.
	\item Fahrzeuge der Klasse Transporter dürfen erst ab einem Alter von 25 Jahren gemietet werden.
	\item Fahrzeuge der Klasse SUV dürfen erst ab einem Alter von 30 Jahren gemietet werden.
	\item Für Fahrzeuge der Klasse Standard gibt es keine zusätzlichen altersabhängigen Einschränkungen über die Mindestanforderung von 21 Jahren hinaus.
\end{enumerate}

\section*{Äquivalenzklassen}

\textbf{Dimension Alter:}
\begin{itemize}
	\item ÄK-A1: Alter $< 21$ (ungültig für alle Klassen)
	\item ÄK-A2: $21 \leq$ Alter $< 25$ (gültig nur für Standard)
	\item ÄK-A3: $25 \leq$ Alter $< 30$ (gültig für Standard und Transporter)
	\item ÄK-A4: Alter $\geq 30$ (gültig für alle Klassen)
\end{itemize}

\textbf{Dimension Führerscheinbesitz:}
\begin{itemize}
	\item ÄK-F1: Führerschein $< 2$ Jahre (ungültig)
	\item ÄK-F2: Führerschein $\geq 2$ Jahre (gültig)
\end{itemize}

\textbf{Dimension Fahrzeugklasse:}
\begin{itemize}
	\item ÄK-K1: Standard (Mindestalter 21)
	\item ÄK-K2: SUV (Mindestalter 30)
	\item ÄK-K3: Transporter (Mindestalter 25)
\end{itemize}
