\chapter*{Aufgabe 1: Lohnsteuerberechnung}

Für die Berechnung der Lohnsteuer gelten folgende Regeln:
\begin{itemize}
	\item Bruttolohn $< 1000$ $\rightarrow$ keine Lohnsteuer
	\item Bruttolohn $1000$--$2500$ $\rightarrow$ 12\,\% Lohnsteuer
	\item Bruttolohn $> 2500$ $\rightarrow$ 15\,\% Lohnsteuer
\end{itemize}

\section*{Pseudocode}

\begin{lstlisting}[language={},numbers=left,xleftmargin=2em]
FUNCTION BerechneLohnsteuer(Nettolohn)
  IF (Nettolohn < 1000) THEN
    Steuer = 0
  ENDIF
  IF (Nettolohn >= 1000 UND Nettolohn <= 2500) THEN
    Steuer = Nettolohn * 0,12
  ENDIF
  IF (Nettolohn > 2500) THEN
    Steuer = Nettolohn * 0,15
  ENDIF
  RETURN Steuer
END FUNCTION
\end{lstlisting}

\chapter*{Aufgabe 2: Rabattberechnung}

Die Funktion berechnet einen Rabatt basierend auf Warenkorbwert, Rabattcode und Gültigkeitsdatum.

\section*{Pseudocode}

\begin{lstlisting}[language={},numbers=left,xleftmargin=2em]
FUNCTION berechneRabatt(warenkorbwert, rabattcode, gueltig_bis)
  heute = aktuelles Datum
  IF (warenkorbwert >= 50 UND rabattcode == "SOMMER20"
      UND heute <= gueltig_bis) THEN
    rabatt = 0.1
  ELSE
    rabatt = 0
  ENDIF
  RETURN rabatt
END FUNCTION

\chapter*{Aufgabe 3: Bubble-Sort}

\section*{Pseudocode}

\begin{lstlisting}[language={},numbers=left,xleftmargin=2em]
FUNCTION BubbleSort(arr)
  n = LENGTH(arr)
  FOR i FROM 1 TO n-1 DO
    FOR j FROM 1 TO n-i DO
      IF arr[j] > arr[j+1] THEN
        SWAP(arr[j], arr[j+1])
      END IF
    END FOR
  END FOR
END FUNCTION
\end{lstlisting}
