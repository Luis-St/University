## Lösungsweg

### a) Faktorfunktionen und maximal erreichbare Produktion

**Prozess I** (Koeffizienten a₁ᴵ = 5, a₂ᴵ = 2):

Faktorfunktionen:
- r₁ᴵ = 5·x
- r₂ᴵ = 2·x

Maximale Produktion (Minimum der Quotienten r̄ᵢ / aᵢ):

x^I = min{ r̄₁ / a₁ᴵ ; r̄₂ / a₂ᴵ } = min{ 1.100 / 5 ; 1.000 / 2 } = min{ 220 ; 500 } = **220**

→ Engpass ist **Faktor 1** (kleinster Quotient).

**Prozess II** (Koeffizienten a₁ᴵᴵ = 3, a₂ᴵᴵ = 4):

Faktorfunktionen:
- r₁ᴵᴵ = 3·x
- r₂ᴵᴵ = 4·x

Maximale Produktion:

x^II = min{ r̄₁ / a₁ᴵᴵ ; r̄₂ / a₂ᴵᴵ } = min{ 1.100 / 3 ; 1.000 / 4 } = min{ 366,67 ; 250 } = **250**

→ Engpass ist **Faktor 2** (kleinster Quotient).

### b) Darstellung im (r₁; r₂)-Diagramm

- **Prozessstrahlen:** Jeder Prozess wird durch einen vom Ursprung ausgehenden Prozessstrahl dargestellt. Die Steigung ergibt sich aus dem Quotienten a₂ / a₁:
  - Prozess I: Steigung a₂ᴵ / a₁ᴵ = 2 / 5 = **0,4**
  - Prozess II: Steigung a₂ᴵᴵ / a₁ᴵᴵ = 4 / 3 ≈ **1,33**
- **Faktorrestriktionen:** Die Beschränkungen r₁ ≤ 1.100 und r₂ ≤ 1.000 erscheinen als senkrechte (r₁ = 1.100) bzw. waagerechte (r₂ = 1.000) Begrenzungslinien, die den zulässigen Bereich einschließen.
- **Isoquanten:** Die Isoquante zum jeweils ermittelten Outputniveau ist der rechtwinklige Eckpunkt (Leontief-Isoquante) auf dem Prozessstrahl beim Engpass. Effizient ist ausschließlich dieser Eckpunkt.
  - Prozess I: Eckpunkt bei x^I = 220 → r₁ᴵ = 5·220 = 1.100 (Engpass ausgeschöpft), r₂ᴵ = 2·220 = 440.
  - Prozess II: Eckpunkt bei x^II = 250 → r₁ᴵᴵ = 3·250 = 750, r₂ᴵᴵ = 4·250 = 1.000 (Engpass ausgeschöpft).

## Ergebnis

| | Faktorfunktionen | Maximale Produktion | Engpass |
|---|---|---|---|
| **Prozess I** | r₁ᴵ = 5x, r₂ᴵ = 2x | x^I = **220** | Faktor 1 |
| **Prozess II** | r₁ᴵᴵ = 3x, r₂ᴵᴵ = 4x | x^II = **250** | Faktor 2 |

Prozess II erreicht bei den gegebenen Faktorbeständen mit 250 Einheiten das höhere Outputniveau als Prozess I mit 220 Einheiten. Nur der jeweilige Eckpunkt auf dem Prozessstrahl ist effizient; die über den Engpass hinaus verfügbaren Faktormengen bleiben ungenutzt.
