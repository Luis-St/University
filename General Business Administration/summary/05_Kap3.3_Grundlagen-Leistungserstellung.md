## Grundmodell des Leistungserstellungsprozesses

Betrachtet wird ein **Produktionsunternehmen**.
Der Prozess führt von der Beschaffung über die Verarbeitung bis zum Absatz und lässt sich als **Input → Transformation → Output** darstellen.

```
BESCHAFFUNGSMARKT                                          ABSATZMARKT
        │                                                        ▲
        ▼                                                        │
   ┌──────────────── INPUT ────────────────┐                     │
   │ Betriebsmittel                        │                     │
   │   z.B. Maschinen, Anlagen, Werkzeuge  │                     │
   │ Gebäude                               │   ┌───────────┐     │
   │ Werkstoffe                            │──▶│  Be- und  │──▶ Output:
   │   z.B. Roh-, Hilfs-, Betriebsstoffe   │   │  Verar-   │   FE (Fertigerzeugnisse)
   │ Arbeitskräfte                         │   │  beitung  │   HF (Halbfabrikate)
   │   Techn. u. kaufm. Personal,          │   └───────────┘
   │   eigene Mitarbeiter, Fremdarbeiter   │  (Transformation)
   └───────────────────────────────────────┘
        │                                                        │
   Ausgaben / Aufwand / Kosten                    Einnahmen / Ertrag / Erlöse
```

### Input (Produktionsfaktoren)

Vom **Beschaffungsmarkt** bezogen und in den Prozess eingebracht:

- **Betriebsmittel** – z.B. Maschinen, Anlagen, Werkzeuge
- **Gebäude**
- **Werkstoffe** – z.B. Rohstoffe, Hilfsstoffe, Betriebsstoffe
- **Arbeitskräfte** – technisches und kaufmännisches Personal; eigene Mitarbeiter und Fremdarbeiter

### Throughput / Transformation

- **Be- und Verarbeitung** der eingesetzten Faktoren (Faktorkombination).

### Output

- **FE = Fertigerzeugnisse**
- **HF = Halbfabrikate**

Der Output wird über den **Absatzmarkt** verwertet.

### Wertmäßige Betrachtung (Zuordnung Input/Output)

- Auf der **Inputseite**: **Ausgaben**, **Aufwand**, **Kosten**.
- Auf der **Outputseite**: **Einnahmen**, **Ertrag**, **Erlöse**.

## Güterwirtschaftliches Gleichgewicht

- **Leistungsziele** legen fest: **was, wann, wo, in welcher Qualität, in welcher Menge** produziert/abgesetzt wird; Natur: **strategisch / taktisch / operativ**.
- **Kern des Gestaltungsproblems:** Abstimmung von **Beschaffungs-, Herstellungs- und Absatzvorgängen**.
- Gelingt dies → **güterwirtschaftliches Gleichgewicht** (Formen: **stationär** oder **evolutionär**). Wichtig: Gleichgewicht **≠ automatisch optimal** – die **Optimalitätsbedingung leitet sich aus den Erfolgszielen ab**.
- **Optimales güterwirtschaftliches Gleichgewicht:** Ergebnisse stehen im Einklang mit den Leistungszielen **und** sind bestmöglich bzgl. der Erfolgskriterien (Kosten, Ertrag, Gewinn, Rentabilität). Problem ist **dynamisch** (Veränderung im Zeitablauf, alle Phasen, multikriterielle Zielfunktionen).

## Gestaltungsprobleme eines optimalen güterwirtschaftlichen Gleichgewichts

Drei Phasen (betrachtet wird ein produzierendes Unternehmen):

1. **Bereitstellung der Produktionsfaktoren (Beschaffung)**
   - Faktoren in der erforderlichen **Art, Güte und Menge am richtigen Ort** bereitstellen.
   - **Bereitstellungskosten minimieren** unter der Bedingung, dass eine **maximale Produktivität der Faktorkombination** gewährleistet ist.

2. **Phase der betrieblichen Faktorkombination (Produktion)**
   - Realisierung der **Minimalkostenkombination** unter Berücksichtigung der produktionstechnischen Bedingungen.

3. **Absatzphase mit allen vertriebsbezogenen Maßnahmen (Absatz)**
   - Optimierung des Einsatzes der absatzpolitischen Instrumente (**Marketing-Mix**) unter Berücksichtigung
     - der **Produkt- und Marketingkosten** sowie
     - von **Umsatz-, Gewinn-, Rentabilitätszielen**.

## Auf einen Blick

- **Modell:** Input (Produktionsfaktoren) → Transformation (Be- und Verarbeitung) → Output (FE/HF); Beschaffungsmarkt vorn, Absatzmarkt hinten.
- **Inputfaktoren:** Betriebsmittel, Gebäude, Werkstoffe (Roh-/Hilfs-/Betriebsstoffe), Arbeitskräfte.
- **Wertgrößen:** Input = Ausgaben/Aufwand/Kosten; Output = Einnahmen/Ertrag/Erlöse.
- **Leistungsziele** definieren was/wann/wo/Qualität/Menge; Natur: strategisch/taktisch/operativ.
- **Güterwirtschaftliches Gleichgewicht:** Abstimmung von Beschaffung, Herstellung, Absatz; Formen stationär oder evolutionär; ≠ automatisch optimal.
- **Optimalitätsbedingung** stammt aus den Erfolgszielen (Kosten-, Ertrags-, Gewinn-, Rentabilitätskriterien).
- **Drei Gestaltungsprobleme:** (1) Beschaffung – Bereitstellungskosten min. bei max. Produktivität; (2) Produktion – Minimalkostenkombination; (3) Absatz – optimaler Marketing-Mix.
