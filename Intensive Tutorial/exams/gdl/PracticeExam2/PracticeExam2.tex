\section*{Aufgabe 1: Bildverarbeitung -- obere und untere Hälfte vertauschen \hfill (25 Punkte)}

Ein Bitmap werde in einem zweidimensionalen \texttt{int}-Array \texttt{p} gespeichert. Jede Farbe wird durch eine Zahl vom Typ \texttt{int} dargestellt. Die Variable \texttt{p[x][y]} steht für die Farbe des Bildes an der Koordinate $(x,y)$. Der Ursprung des Koordinatensystems liegt links oben, somit enthält \texttt{p[0][0]} die Farbe des Pixels in der linken oberen Ecke.

\begin{center}
\begin{tikzpicture}[font=\scriptsize]
	\foreach \xi in {0,...,5}
		\foreach \yi in {0,...,4}
			\node[draw, minimum width=1.25cm, minimum height=0.7cm]
			at (1.3*\xi, -0.75*\yi) {p[\xi][\yi]};
\end{tikzpicture}
\end{center}

Die Breite des Bildes ist \texttt{p.length}, die Höhe ist \texttt{p[0].length}. Damit liegt $x$ im Bereich $0$ bis \texttt{p.length-1} und $y$ im Bereich $0$ bis \texttt{p[0].length-1}.

Schreiben Sie eine Klassenmethode \texttt{f}, die in dem Bild die \textbf{obere Hälfte mit der unteren Hälfte vertauscht} (Spiegelung von oben und unten). Die Methode \texttt{f} soll als Parameter \texttt{p} einen zweidimensionalen \texttt{int}-Array haben und keinen Rückgabewert. Es darf davon ausgegangen werden, dass die Höhe des Bildes eine gerade Zahl ist.

\section*{Aufgabe 2: Syntaxfehler -- Klasse Statistik \hfill (20 Punkte)}

Der Programmcode der Klasse \texttt{Statistik} enthält mehrere syntaktische Fehler. Geben Sie zu jedem Fehler die Zeilennummer der Zeile an, in der sich der Fehler befindet, und beschreiben Sie jeweils den Fehler.

\begin{lstlisting}
public class Statistik {

    public int anzahl = 0;

    public static int summe(int[] a) {
        int s = 0;
        for (int i = 0; i < a.length; i++)
            s = s + a[i];
        return s
    }

    public static double mittel(int[] a) {
        double m = summe(a) / a.length;
        if (a.length > 0)
            return m;
    }

    public static boolean enthaelt(int[] a, int v) {
        for (int i = 0; i < a.length; i++) {
            if (a[i] == v)
                return true;
        }
        return "nein";
    }

    public static void main(String[] args) {
        int[] x = { 4, 8, 15, 16 };
        int s = summe(x);
        System.out.println(mittel x);
        boolean b = enthaelt(x);
        anzahl = anzahl + 1;
    }
}
\end{lstlisting}

\section*{Aufgabe 3: Programmausgabe -- Schleife über ein Array \hfill (20 Punkte)}

Zu welcher Ausgabe kommt es, wenn man die Methode \texttt{main} der Klasse \texttt{A} ausführt?

\begin{lstlisting}
public class A {

    public static void main(String[] args) {
        int[] x = { 1, 2, 3, 4 };
        for (int i = 0; i < x.length; i++) {
            x[i] = x[i] + x[(i + 1) % x.length];
            System.out.println(x[i]);
        }
    }
}
\end{lstlisting}

\begin{hinweis}[Empfehlung]
	Notieren Sie nach jedem Schleifendurchlauf den vollständigen Inhalt des Arrays \texttt{x}. Beachten Sie, dass jede Zuweisung den Array sofort verändert.
\end{hinweis}

\section*{Aufgabe 4: Methode implementieren -- längstes aufsteigendes Teilstück \hfill (25 Punkte)}

Schreiben Sie eine statische Methode \texttt{maxAufstieg}, die einen Parameter \texttt{a} vom Typ \texttt{int}-Array hat und einen Rückgabewert vom Typ \texttt{int}.

Die Methode soll die Länge des \textbf{längsten zusammenhängenden, streng aufsteigenden Teilstücks} des Arrays bestimmen und als Rückgabewert zurückgeben. Ein Teilstück ist streng aufsteigend, wenn jede Zahl echt größer ist als ihre Vorgängerzahl. Ist der Array leer, so soll der Rückgabewert $0$ sein.
