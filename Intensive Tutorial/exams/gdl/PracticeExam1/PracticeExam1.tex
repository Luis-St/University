\section*{Aufgabe 1: Bildverarbeitung -- Rahmen schwarz färben \hfill (25 Punkte)}

Ein Bitmap werde in einem zweidimensionalen \texttt{int}-Array \texttt{p} gespeichert. Jede Farbe wird durch eine Zahl vom Typ \texttt{int} dargestellt. Die Variable \texttt{p[x][y]} steht für die Farbe des Bildes an der Koordinate $(x,y)$. Der Ursprung des Koordinatensystems liegt links oben, somit enthält \texttt{p[0][0]} die Farbe des Pixels in der linken oberen Ecke.

\begin{center}
\begin{tikzpicture}[font=\scriptsize]
	\foreach \xi in {0,...,5}
		\foreach \yi in {0,...,4}
			\node[draw, minimum width=1.25cm, minimum height=0.7cm]
			at (1.3*\xi, -0.75*\yi) {p[\xi][\yi]};
\end{tikzpicture}
\end{center}

Die Breite des Bildes ist \texttt{p.length}, die Höhe ist \texttt{p[0].length}. Damit liegt $x$ im Bereich $0$ bis \texttt{p.length-1} und $y$ im Bereich $0$ bis \texttt{p[0].length-1}. Der Wert $0$ steht für die Farbe schwarz.

Schreiben Sie eine Klassenmethode \texttt{f}, die einen \textbf{Rahmen der Dicke 15 Pixel} schwarz färbt: Alle Pixel, deren Abstand zum Bildrand weniger als 15 Pixel beträgt (d.\,h.\ die in den äußersten 15 Spalten bzw.\ Zeilen liegen), sollen auf $0$ gesetzt werden. Die Methode \texttt{f} soll als Parameter \texttt{p} einen zweidimensionalen \texttt{int}-Array haben und keinen Rückgabewert.

\begin{lstlisting}
public class Konto {

    public double stand;
    private String inhaber

    public Konto(String name) {
        inhaber = name;
        stand = 0.0;
    }

    public void einzahlen(double betrag) {
        if (betrag > 0)
            stand = stand + betrag;
        return stand;
    }

    public boolean abheben(double betrag) {
        if (betrag <= stand) {
            stand = stand - betrag;
            return true;
        }
    }

    public static void main(String[] args) {
        Konto k = new Konto("Anna");
        k.einzahlen(100);
        double rest = k.abheben(30);
        System.out.println(k.stand);
        if (k.stand > 0)
            System.out.println("Guthaben: " k.stand);
    }
}
\end{lstlisting}

\section*{Aufgabe 3: Programmausgabe -- rekursive Methode \hfill (20 Punkte)}

Zu welcher Ausgabe kommt es, wenn man die Methode \texttt{main} der Klasse \texttt{X} ausführt?

\begin{lstlisting}
public class X {

    public static void f(int n) {
        if (n <= 0)
            return;
        System.out.print(n + " ");
        f(n - 1);
        System.out.print(n + " ");
    }

    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {
            f(i);
            System.out.println();
        }
    }
}
\end{lstlisting}

\begin{hinweis}[Hinweis]
	\texttt{System.out.print} erzeugt eine Ausgabe ohne anschließenden Zeilenumbruch, \texttt{System.out.println()} erzeugt einen Zeilenumbruch.
\end{hinweis}

\begin{hinweis}[Empfehlung]
	Bestimmen Sie zunächst getrennt die Ausgabe der Aufrufe \texttt{f(1)}, \texttt{f(2)} und \texttt{f(3)} und setzen Sie diese anschließend zur Gesamtausgabe zusammen.
\end{hinweis}

\section*{Aufgabe 4: Methode implementieren -- Nachbarsummen \hfill (25 Punkte)}

Die Methode \texttt{f} soll jeden Wert des Arrays durch die \textbf{Summe seiner beiden direkten Nachbarn} ersetzen. Das erste und das letzte Element des Arrays haben nur einen Nachbarn und sollen \textbf{unverändert} bleiben.

\begin{hinweis}[Hinweis]
	Überlegen Sie, weshalb Sie zur korrekten Berechnung eine Kopie der ursprünglichen Werte benötigen.
