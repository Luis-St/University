# Aufgabe B – C-Programm erstellen

Erstellen Sie ein kurzes C-Programm, das die im selben Ordner liegende Datei `helloworld.c` öffnet, ihren Inhalt liest und auf der Konsole ausgibt. Anschließend soll die Datei wieder geschlossen werden.

Verwenden Sie zum Öffnen, Lesen und Schließen die Systemaufrufe `open`, `read` und `close`.
