# Aufgabe B – C-Programm erstellen

Erstellen Sie ein kurzes C-Programm mit zwei Funktionen `istGerade` und `istUngerade`, die sich gegenseitig aufrufen und ohne Verwendung des Modulo- (`%`) oder Divisionsoperators (`/`) bestimmen, ob eine nichtnegative ganze Zahl gerade ist.

Prüfen Sie im Hauptprogramm den Wert `42` und geben Sie aus, ob die Zahl gerade oder ungerade ist.
