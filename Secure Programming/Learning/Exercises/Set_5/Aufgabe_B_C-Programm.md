# Aufgabe B – C-Programm erstellen

Erstellen Sie ein kurzes C-Programm, das einen zusammengesetzten Datentyp für einen Ringträger mit den Feldern Vorname, Nachname und Spitzname definiert.

Legen Sie ein Array mit zwei Ringträgern an, initialisieren Sie es, und geben Sie in einer Schleife für jeden Ringträger den Spitznamen aus.
