# Aufgabe A – Assembly → C

Sie haben aus einem fremden Programm ein Fragment Assembly extrahiert (Datei `fragment.s`, x86-64, AT&T-Syntax, Linux). Im Rahmen einer Präsentation möchten Sie den Ablauf für Teammitglieder erläutern, die kein Assembly lesen.

Erläutern Sie den Ablauf des Assembly-Programms. Beschreiben Sie Schritt für Schritt, was der Code tut, und bestimmen Sie, mit welchem Wert das Programm endet.
