# Aufgabe B – C-Programm erstellen

Erstellen Sie ein kurzes C-Programm mit einer Funktion `swap`, die die Werte zweier `int`-Variablen vertauscht.

Deklarieren Sie im Hauptprogramm zwei Variablen `a = 2` und `b = 4`, rufen Sie `swap` auf und geben Sie anschließend beide Werte aus. Die Vertauschung muss nach dem Aufruf im Hauptprogramm sichtbar sein.
